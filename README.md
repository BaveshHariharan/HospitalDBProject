# HospitalDBProject
TIP:
This project contains:
- SQL schema (tables, primary keys, foreign keys)
- Sample data inserts
- Triggers
- Views for reports and analytics

Database used: SQLite

To run:
1. Open any SQLite editor
2. Execute the SQL script in hospital.sql
3. Use the sample queries to test

### Tip
If your database gives errors, drop all tables first using:
DROP TABLE IF EXISTS tableName;
